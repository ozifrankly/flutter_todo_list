# Depencies

## Install

- [Dart](https://dart.dev/get-dart)
- [Slidy](https://github.com/Flutterando/slidy)
- [Flutter](https://flutter.dev/docs/get-started/install)


## Run first time

Run this command to slidy generate ``the`` **.g.dart** files.
```bash 
slidy run mobx_build
```

Run the flutter.
```
flutter run
```
